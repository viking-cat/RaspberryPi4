## Download and Install Instructions

```bash
# Download SDL2
$ wget https://www.libsdl.org/release/SDL2-2.0.9.tar.gz -O SDL2-2.0.9.tar.gz

# Unpack SDL2-2.0.9.tar.gz file
$ tar -xvzf SDL2-2.0.9.tar.gz
```
